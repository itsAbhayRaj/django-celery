# Celery Email Project

Advanced Django Course - Celery & Redis

This project demonstrates advanced Django concepts using Celery for asynchronous task processing and Redis as the message broker.

## Features

- Asynchronous email sending using Celery
- Task scheduling and management
- Redis as message broker
- Django REST API for email operations
